# Summer Vacation Coding Assignments 2026

## 📚 Overview

This repository contains my solutions to the **Summer Vacation Self-Learning Assignments (June–July 2026)** assigned by the Career Development Centre, GL Bajaj Institute of Technology and Management.

The repository includes **120 coding problems** ranging from basic programming concepts to advanced problem-solving and mini-project development. These programs were completed to strengthen programming fundamentals, logical thinking, and coding interview preparation.
